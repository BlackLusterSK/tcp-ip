## Resumen
There are two ways to add data in Firestore.

## Modelo OSI
Create model object and add values to it and `save()` the model. After saving model **model id** and 
**model key** is attached with model object.

## Modelo TCP-IP
Create model object and add values to it and `save()` the model. After saving model **model id** and 
**model key** is attached with model object.
